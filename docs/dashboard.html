Workflow

Step 1
Repository cloned and local Python virtual environment created.
Step 2
Baseline packages installed: NumPy, SciPy, Matplotlib, Jupyter; then CMake, PyTorch, scikit-learn.
Step 3
CMake config modernized to use discovered Python3/NumPy instead of hardcoded legacy paths.
Step 4
Executable rebuilt in build_local/ and run against Python workflow in build/.
Step 5
Runtime debugging and ML migration completed: TensorFlow pipeline replaced with a Torch stacked-LSTM in build/ml_module.py.
Step 6
LSTM tuned for lower rollout error using pushforward-style autoregressive training, scheduler, and gradient clipping.
Step 7
Two deployment variants were trained and compared: one-step input (seq_len=1) and multistep input (seq_len=8), both deployed autoregressively.
Step 8
Forecasting upgraded to a seq2seq encoder-decoder LSTM with curriculum rollout and scheduled sampling to better match autoregressive deployment.
Step 9
Comparison plots were added across model generations: baseline LSTM, stabilized delta-LSTM, and seq2seq LSTM.
Step 10
Added a Koopman-style neural state-space model with stable latent linear dynamics and nonlinear residual correction for direct comparison.
